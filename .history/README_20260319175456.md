# Finance

### Um aplicativo Android para controle de finanças pessoais de forma simples e direta em react-native.

## Objetivo

#### Ensinar aos alunos do curso **Bacharelado em Ciência da Computação** do IF Sudeste MG - _Campus_ Rio Pomba, da disciplina Tópicos em Ciência da Computação III em 2026, como desenvolver uma aplicação usando React Native com autenticação em banco de dados Supabase.

#### Técnicas que serão mostradas:

- organização de pastas no app
- criação de componentes
- rotas etc.
