# Finance

### Um aplicativo para controle de finanças pessoais de forma simples e direta.

## Objetivo

#### Ensinar aos alunos do curso **Bacharelado em Ciência da Computação** do IF Sudeste MG - _Campus_ Rio Pomba, da disciplina Tópicos em Ciência da Computação, como desenvolver uma aplicação usando React Native com autenticação em banco de dados Supabase.

#### Para esse fim, diversas outras técnicas serão mostradas, como organização de pastas no app, criação de componentes, rotas etc.
